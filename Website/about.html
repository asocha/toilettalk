What is Toilet Talk?

ToiletTalk’s goal is to help users find and travel to restrooms with ease. It is designed with the android mobile OS as the primary platform with a website that will implement much of the same functionality as the Android platform, with a few changes. Those changes include greater functionality with the Google Maps interface. On both platforms, users will be able to search for restroom by location and rating, as well as using their phone’s GPS to find nearby restrooms. Users can then create reviews for these restrooms and rate them. We also include route planning which is similar to Google Maps except that we will also display all restrooms along the route, within a certain distance of the road. Users can save these routes to their profiles, as well as saving their favorite individual restrooms. Overall, we hope ToiletTalk will become a quick and easy to use application that anyone can use to find a restroom.
